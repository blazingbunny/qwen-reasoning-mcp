# Qwen Reasoning MCP Server

A lightweight reasoning engine for AI agents using Qwen2.5-Coder-0.5B-Instruct.

## Features
- **MCP Native**: Automatically exposes reasoning tools via Model Context Protocol.
- **Fast**: Optimized for HF Free Tier (CPU/T4).
- **Multilingual**: Supports 29+ languages.

## Usage
Add this Space to your MCP client (Cursor, Claude Desktop, etc.) using the URL or the MCP badge on Hugging Face.
